# Product Management System (Ruby on Rails)

A scalable Ruby on Rails application for managing products, categories, and inventory with RESTful APIs.

## Tech Stack

- Ruby 3.x
- Rails 7.x
- PostgreSQL
- Redis (for caching / Sidekiq if used)
- Sidekiq (optional background jobs)
- RSpec (testing)


## Features

- Product CRUD (Create, Read, Update, Delete)
- Category management
- Inventory tracking
- Search & filtering
- Pagination
- API support (JSON)
- Background jobs (optional)
- CSV import/export (optional)
